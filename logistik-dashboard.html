📋 Panduan Setup Google Apps Script
Langkah 1: Buka Google Spreadsheet kamu
Langkah 2: Klik menu Extensions → Apps Script
Langkah 3: Hapus kode yang ada, paste kode Apps Script dari file apps-script.gs yang disertakan
Langkah 4: Klik Deploy → New Deployment → Web App
Langkah 5: Set "Who has access" → Anyone
Langkah 6: Copy URL yang muncul, paste di field URL di atas
Langkah 7: Klik Simpan Konfigurasi → Test Koneksi
